# Aglaia

Première version gratuite d’Aglaia sous forme d’application web mobile locale.

## Ouvrir l’application

Double-cliquer sur `index.html`. L’application s’ouvre dans le navigateur et fonctionne sans compte ni serveur.

Pour profiter de l’installation PWA et du mode hors connexion, servir ensuite le dossier avec un hébergement HTTPS ou un serveur web local.

## Fonctionnalités

- navigation entre Accueil, Dressing, Ajouter, Tenues, Impact et Profil ;
- ajout de vêtements avec photo et emplacement détaillé ;
- recherche et filtres par catégorie ;
- consultation, modification et suppression ;
- création et suppression de tenues ;
- préférences de profil ;
- sauvegarde locale dans le navigateur.

Les données restent sur l’appareil et peuvent être effacées si les données du navigateur sont supprimées.
